# SDG API

A Flask RESTFUL API built to retrieve Sustainable Development Goals of world countries !

# How to install

Create a virtual environment :  `python3 -m venv venv` in app repo
Launch the virtual environment : `. .venv/bin/activate` or `. virtualenv.sh`
Install prerequisites in virtual environment : `pip install -r requirements.txt`

# How to launch the app

Go to your virtual environment : `python3 api.py`

# How to quit virtual env

Use `deactivate`
